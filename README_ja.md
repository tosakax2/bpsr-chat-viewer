BPSR Chat Viewer は、ゲーム「**Star Resonance**」のゲーム内チャットをリアルタイムでキャプチャし、読みやすい外部ウィンドウに表示するスタンドアロンのデスクトップアプリケーションです。ゲームクライアントを改変することなく、独立したGUIからチャットを確認できます。

## 主な機能

* **リアルタイムチャットキャプチャ**: ネットワークパケットをスニッフィングし、チャットメッセージをリアルタイムで取得します。
* **メッセージを見逃さない**: ゲーム内チャットとは異なり、クラフト画面やロード画面などのチャットが開けない場面でもシームレスに機能し続けるため、重要なメッセージを見逃すことがありません。
* **統合インターフェース**: ワールド、ギルド、チャンネル、パーティ、システムなど、異なるチャンネルのメッセージを1つの画面で閲覧できます。
* **チャンネルフィルター**: 特定のチャンネルのみを表示・非表示にすることで、重要な会話に集中できます。
* **ポップアウトウィンドウ**: 特定のチャンネル専用の独立したウィンドウを別画面として開くことができます。
* **通知音**: メッセージ受信時のサウンド通知機能（チャンネルごとの音の違い、音量調節機能付き）。
* **スタンドアロン実行可能**: 環境構築不要で、簡単に利用できる `.exe` 形式として配布しています。

## ダウンロードとインストール

最も簡単な使用方法は、事前にビルドされた実行ファイルをダウンロードすることです。

### ステップ 1: Npcap のインストール (必須)
ネットワーク上のチャットパケットをキャプチャするため、パケットキャプチャ用のシステムドライバが必須となります。
1. [Npcap 公式サイト](https://npcap.com/#download) からインストーラーをダウンロードします。
2. インストーラーを実行してインストールを完了させます（インストールオプションはすべてデフォルトのままで問題ありません）。

### ステップ 2: アプリの起動
1. このリポジトリの [Releases](../../releases) ページにアクセスします。
2. 最新の `.zip` リリースをダウンロードします。
3. 任意の場所にフォルダを展開（解凍）します。
4. フォルダ内にある `BPSRChatViewer.exe` を実行します。

> [!NOTE] 
> アプリケーションの性質上、起動時に管理者権限を求めるUAC（ユーザーアカウント制御）ダイアログが表示されます。これは正常な動作であり、Npcapと連携してスニッフィングを行うために必須の権限です。

## ソースからのビルド

ソースコードから直接実行したい場合や、自身で実行ファイルをビルドしたい場合は以下の手順に従ってください。

### 前提条件
* **Python** (`uv` パッケージマネージャー経由)
* **Node.js** (フロントエンド用)
* **Npm**
* **Npcap / WinPcap** (Windowsでのパケットキャプチャのために必須です)

### 1. ソースコードから直接実行する
同梱されているバッチファイルを使用して直接GUIを起動できます：
```bat
run_gui.bat
```
このスクリプトは、自動的に管理者権限のチェック、フロントエンド（React）のビルド、Pythonの仮想環境（`.venv`）の構築とパッケージインストールを行い、アプリを起動します。

### 2. 実行ファイル (exe) のビルド
アプリを配布可能なスタンドアロンフォルダにパッケージングするには以下を実行します：
```bat
build_exe.bat
```
ビルド完了後、`backend/dist/BPSRChatViewer` に実行ファイルを含むフォルダが生成されます。このフォルダごと配布することができます。
